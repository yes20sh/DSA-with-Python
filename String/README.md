10. **Pangram Check**

    - Last position
    - Kth position

12. **Remove Character**  
    Remove a character from:
    - First position
    - Last position
    - Kth position

13. **Substring Search**  
    Find whether a specific substring exists within a string.

14. **Generate Substrings**  
    Generate and print all possible substrings of a string.
